# Bot Network (Python)

Una red modular de bots diseñada para demostrar comunicación `publish/subscribe`,
simulación de sensores y análisis automático de datos.

##  Características principales

✔ Arquitectura tipo microservicios  
✔ Sistema de mensajería interno (MessageBus)  
✔ Bots que se comunican entre sí  
✔ Simulación de datos IoT  
✔ Generación automática de alertas  
✔ Extensible para IA, automatización o sistemas distribuidos
